If you prefer a **local-first** approach:


2. Create a new GitBook project:
   ```bash

   gitbook init
   ```

   ```bash
   gitbook serve
   ```

  ```json
  {
    "plugins": [

      "search", 
      "anchors"
  ```

- Add a deploy script in `package.json`:
  ```json
  "scripts": {
    "deploy": "gitbook build && gh-pages -d _book"
  }
  ```
- Run:
  ```bash
  npm run deploy
  ```
